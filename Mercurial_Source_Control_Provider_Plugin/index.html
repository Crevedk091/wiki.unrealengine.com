# Mercurial Source Control Provider Plugin

Name

Mercurial Source Control Provider Plugin

Category

Editor.Source Control

Author

Vadim Macagon

Version

0.3 (alpha)

UE4 Build

4.3.X

View on GitHub

Discuss in the Forums

This plugin allows the Unreal Editor to interact with the Mercurial DVCS. It supports basic add, remove, revert, commit operations, provides access to asset history, and enables built-in visual diffing of Blueprints. Installation instructions and downloads can be found on the project's GitHub page.
