Who

I am a veteran modder who has been in and out of the community for many years, without a single released title to my name with so much experience I feel I am heading towards a pinnacle, hopefully not a precipice.

What

Unreal Engine

Editor

  • Game mode (G) for all previews.
  • ESC from F11 fullscreen should go back to windowed.
  • SubComponent Properties - Skeletal and Staticmeshes can be expanded and have their materials changed in their properties object (component editor/etc).

Blueprints

  • Probe Nodes / Bridges (Macros)
  • Macro Libraries should show categories from context (rmb) menu.
  • Variable that can be Snapped to the node they are connected to.
  • Convert Enum (Byte) to Int automatically when connecting it to an Int pin in Blueprints Graph Editor.
  • Not equal Enumerator node.
  • Blueprints in the details panel should show find in Content Browser like Materials/Meshes.
  • Make Spawn Node more like Trace node with Spawn Failed/bool outputs.

Geometry

  • Convert BSP additive/subtractive to collision volume.

Project Castor

The project is a fast paced first person arena game with a fantasy theme, it aims to appeal to a wide audience and will hopefully be welcomed by more competitive gamers looking for deeper elements of strategy and team builds in a shooter type environment.

Tutorials

My goal is to really aim at people like I was when I was 14, I was a gamer, I had no clue about what actually went on inside a game engine in terms of design/programming flow but I did have a grasp of some basic terminology and a solid understanding of math and logic.

When

I began my journey with Unreal Engine some time ago, initially I explored Unreal Tournaments tools but found them unstable on my system which kept me from really exploring things. At the time I was still a beginner at C having only been using Quake engine from a programming standpoint so UnrealScript wasnt really something I got into. I started to explore Level Design and for some reason the subtractive methodology which was the only one available in Unreal Engine escaped my comprehension but as I worked with Worldcraft and Quark with Quake engine my knowledge of BSP grew and I was able to get better at using Unreal Tournament editor. I actually found using 3D space much easier than 2D in editors for Doom or Duke2D (Build), I eventually transitioned to 3DSMax on the news of Epic having a new easier content pipeline in the works for UT2003 and so it all began.

Where

Perth, Australia

Why

Ever since I began with MapEdit.exe for Wolf3d all those years ago I started with this idea that I wanted to contribute to gaming that it was about more for me than getting to the next game. Eventually I started dabbling with paint programs, making skins for reaper bots and sprites in build, I had a technical flair too being able to understand Hex and edit memory to get additional stats in RPG's because of this I naturally formed a love for games because of this blend of both creative and technical fields.
